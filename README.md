# analisis-datos-statgraphics — Análisis estadístico de datos

![Statgraphics](https://img.shields.io/badge/Statgraphics-19-blue)
![Estadística](https://img.shields.io/badge/estad%C3%ADstica-descriptiva%20e%20inferencial-green)

Trabajos de **análisis estadístico** con Statgraphics sobre conjuntos de datos
reales: desde la descripción de las variables hasta la inferencia y el contraste
de hipótesis, con la interpretación de los resultados recogida en informes.

## Contenido

- **`estadistica-descriptiva/`** — estudio descriptivo de variables (centralización, dispersión, gráficos).
- **`regresion-lineal/`** — modelos de regresión para relacionar dos variables (p. ej. alfabetización y mortalidad infantil).
- **`comparacion-de-dos-variables/`** — intervalos de confianza y contraste de hipótesis sobre medias y varianzas de muestras independientes.
- **`datos/`** — conjuntos de datos utilizados (`Mundo 2013`, `Salarios`).

## Competencias que pone en práctica

- **Estadística descriptiva** e **inferencial** (intervalos de confianza, contrastes, normalidad).
- **Regresión** y análisis de la relación entre variables.
- Interpretación de resultados y **redacción de informes** técnicos.
